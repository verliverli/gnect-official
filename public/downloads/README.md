# GNECT Android App Downloads

This directory contains the Android APK generated by Microsoft PWA Builder.

## How to generate a new APK:
1. Go to https://pwabuilder.com
2. Enter `https://gnect.vercel.app` as the PWA URL
3. Click "Package for Android"
4. Download the generated APK
5. Place it here as `gnect.apk`
6. Update the version in `download-info.json`
7. Push to GitHub — Vercel will deploy automatically

## File naming:
- `gnect.apk` — the main APK file (users download this)
